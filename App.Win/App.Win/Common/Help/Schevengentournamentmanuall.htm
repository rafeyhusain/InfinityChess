Introduction

  • This document describes how to tournament director could organize a Scheveningen Tournament on InfinityChess. Only admins or users having rank Bishop or higher could organize Tournaments.

  • Following are the steps to create Scheveningen Tournament.

    • 1. How to Play Scheveningen Tournament

    Step 1 - Create Team’s for Scheveningen Tournament

  • Select File > Tournament > Create Team menu to create new team as shown below.

  • Type team name and press Save button as shown below.

    • Step 2- Create Scheveningen Tournament

  • Select File > Tournament > New Tournament menu to create new tournament as shown below.

    • Step 3 – Select Team’s

  • Go Team Tab to select teams for Scheveningen tournament as shown below.

  • By clicking New button you can select teams for tournament.

  • After choosing teams press Select button to include selected teams in tournament as shown below.

  • By clicking New button you can select player from the list of online players or All players.

    • Step 4 – Approve wantin player’s request

  • Select WantIn tab in tournament detail and either approve or decline WantIn players by selecting player and press Approved or Decline button as shown below.

  • Approved players are moved in Team tab according to their chosen team. While declined player will remain in Wantin Player tab.

    • Step 5 – Team Tab

  • Select Team tab to see the distribution of player according to their chosen team as shown below.

    • Step 6 – Start Tournament

  • Click Start Tournament button to start tournament like previous methods.

    • Step 7 – Scheveningen Tournament Result

  • Please select Results tab to see the results.
