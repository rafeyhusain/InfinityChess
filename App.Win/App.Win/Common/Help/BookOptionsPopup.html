Book Option

  • Use book option to determines whether the openings book will be used.

  • Tournament book option is checked(or on) the moves not play that has been excluded from the tournament.

  • Minimum games tells the program how many games to be played.

  • Up to move number determines how long it should be in book.

  • Optimize sets the parameters and their original values, for optimize playing strength.

  • Normal sets for informal games.

  • Handicap sets games levels to the handicap.
