Introduction

  • This document describes how tournament director could organize a Knockout Tournament on InfinityChess. Only admins or users having rank Bishop or higher could organize Tournaments.

  • Following are the steps to create Knockout Tournament.

    • How to Play Knockout Tournament

    Step 1 - Create Knockout Tournament

  • File > Tournament > New Tournament menu to create new team as shown below.

  • Make sure to select “Allow Tie Break” option this option allow you to create tie breaker matches

    • Step 2 – Approve wantin player’s request

  • Select WantIn Player tab in tournament detail and either approve or decline WantIn players by selecting player and press Approved or Decline button as shown below.

    • Step 3 –Register Players

  • In Register Player tab Tournament director have a choice to register other players by clicking Add Player button as shown below.

  • Click Ok button after choosing players from list as shown below.

    • Step 4 – Start Tournament

  • Click Start Tournament button to start tournament like previous methods.

    • Step 5 – Knockout Tournament Matches

  • After finishing Matches Next Match Automatically Created.

  • Tournament Director have to create next round after finish a round.

  • Complete Matches of a Knock Out Tournament.

    Step 6 - Knock Out Tournament Result

  • Please select Standings tab to see the results.
