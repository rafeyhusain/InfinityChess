Introduction

  • This document describes how tournament director could organize a Knockout Tournament on InfinityChess. Only admins or users having rank Bishop or higher could organize Tournaments.

  • Following are the steps to create Knockout Tournament.

    • How to Play Knockout Tournament

    Step 1 - Create Knockout Tournament

  • File > Tournament > New Tournament menu to create new team as shown below.

  • Make sure to select “Allow Tie Break” option this option allow you to create tie breaker matches

    • Step 2 – Approve wantin player’s request

  • Select WantIn tab in tournament detail and either approve or decline WantIn players by selecting player and press Approved or Decline button as shown below.

    • Step 3 –Register Players

  • In Register Player tab Tournament director have a choice to register other players by clicking New button as shown below.

  • Click Select button after choosing players from list as shown below.

  • You can select list of online player & All Player.

    • Step 4 – Start Tournament

  • Click Start Tournament button to start tournament like previous methods.

    • Step 5 – Draw Match in Knockout Tournament

  • After finishing Matches if a match draws then click Draw button in Match Status Column.

    • Step 6 – Tie-Break Match

  • After clicking on Draw button a window appear as shown below.

  • Select White & Black side for players and set up the time and click Save button

  • After clicking Save button a challenge goes to players challenge window now they can play match by accepting challenge.

  • After finishes that match we have a final winner of this Knockout tournament.

  • Click Finish Tournament button to finish the tournament.

    • Step 7 - Knock Out Tournament Result

  • Please select Results tab to see the results.
