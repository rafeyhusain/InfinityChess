Playing Mode

  • Computer:In this option if you want to play with engine select define engine option.

  • Centaur:In this option you play manual to engine.

  • Clock:In this option Time option(in minutes) and Gain per move(in second) to play game.

  • Automatic Challenges/Accepts:In this option you can do challenges and accept challenges automatically.

  • Send Evalution:In this option you can send evaluations with moves in Engine vs Engine.

  • Expected Moves:In this option you can send expected move with moves in Engine vs Engine.
