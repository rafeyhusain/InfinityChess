Game Data:

  • White/Black:In this option enter the name or change the name.

  • Tournament:In this option shown the type of game.

  • ECO Code:Write ECO code in this option.

  • Elo White/Elo Black:In this option set Elo rating of game players.

  • Result:In this option set result of the game.

  • Reset:Clears all data.

  • Also set Date Time of the game.
